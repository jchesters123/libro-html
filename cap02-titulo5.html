Formularios

Formularios <form>

Los formularios son un elemento fundamental para el desarrollo web,ya que son las etiquetas responsables de extraer los datos de un usuario(que se localiza en lo que llamamos cliente), para enviarlos a un sistema encargado de administrar los datos totales que surgan a partir del total de usuarios con la página (que es lo que denominamos servidor)
Para los usuarios se usa la etiqueta <form>.

Los atributos que permiten hacer esto son

  • action "URL": Contiene la URL del servidor que recibe los datos del cliente

  • method: Coniste para indicarle al protocolo de http la forma de envío de los datos del cliente al servidor. Las dos formas de envío son

    • GET: Envía la información a la URL especificada sin encriptar los datos, los cuales permanecen visibles, admite un tráfico de máximo 500 bytes

    • POST: Envía paquetes de datos con máyor peso, encripta los datos y por tanto, los datos permaneceran no visibles en el tráfico

Para poder ver los atributos en funcionamiento, veamos el siguiente ejemplo

            <form action="/servidor/" method="GET">
                <label>¿Quieres trabajar conmigo?</label>
                <input type="mail"/>
                <bottom type="bottom">enviar</bottom>
            </form>

Botones de envío de datos <input>

Una práctica bastante generalizada en el mundo del desarrollo web, es usar las etiquetas <form> de la mano con las etiquetas <label> e <input>, los botones de input son formatos de envío de datos de un cliente a un servidor. Existen una gran cantidad de atributos: envío de datos, selección en listas no numeradas, botones de texto, de numeros, de email, etc. A continuación, vamos a ver a las etiquetas más usadas. Para poder aplicar los ejemplos de las etiquetas, vamos las siguientes reglas de estilo en CSS 

        form, fieldset
        {
        display: flex;
        flex-flow: row wrap
        width: 480px;    
        }

        label
        {
        width: 200px;    
        }
        input
        {
        width: 250px;
        margin-bottom: 7px;    
        }

Etiquetas de recopilación de datos tel, email, url, buscadores

Existen otros atributos en la etiqueta input para recopilar información, tales como los de email, teléfono, y busqueda. Para los casos en específico de email y URL, hay otros atributos esepcíficos como lo ameritan la recolección de esta clase de datos. El código sería

        <form action="/servidor/" method="POST">
            <label>Email: </label><input type="email">
            <label>URL: </label><input type="url">
            <label>Telefono: </label><input type="tel">
            <label>Busqueda: </label><input type="search">
        </form>

Casillas de opciones input type="radio"

Las casillas de opciones, las cuales se activan con el atributo radio funcionan de la misma forma que las casillas de verificación, pero en el caso de las casillas de opciones, solo se puede elegir un solo botón de entre todos. Esto quiere decir que la elección del botón define a la etiqueta label 

        <form>
            <label>opcion a: </label><input type="radio">
            <label>opcion b: </label><input type="radio">
            <label>opcion c: <label><input type="radio">
            <label>opcion d: </label><input type="radio">
        </form>

Botones de input input type="button"

Los formatos de botones de la etiqueta input permiten varias opciones. Hay etiquetas con botones de imagen, con botones de reinicio de formulario, y de envío directo de datos. Los tipos de botones son:

  • <input type="submit">: Sirve para enviar los datos de forma directa al servidor

  • <input type="reset">: Sirve para reiniciar el formulario

  • <input type="image" src="imagen/jpg">: Carga una imagen en el boton-password

Veamos el siguiente ejemplo

        <label>Botón enviar</label>
        <input type="submit" value="enviar">
        <label>Botón enviar con imagen</label>
        <nput type="image" src="boton.jpg">
        <label>Botón resetear</label>
        <input type="reset" value="reiniciar">
        <label>Botón</label>
        <input type="button" value="un boton">

Números y rangos

La etiqueta <input type=number> permite enviar datos numéricos, este etiqueta tiene atributos por defecto que implica escribir un numero entre un rango de números. Estos atributos son min, max y step correspondientemente. para poder tener esto más claro veamos un ejemplo 

        <form>
            <label>Escribe un numero, de 0 a 10</label>
            <input type="number" min=0 max=10>
        </form>
