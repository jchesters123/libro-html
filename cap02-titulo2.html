La cabecera de html

La cabecera <head>

La cabecera <head><head/>es la primera etiqueta que se ubica en un documento de html, y contiene todos los elementos que rigen al documento en terminos generales: los metadatos, las librerías, los link de css y javascript, y el título. Los editores de codigo (tales como Atom o Visual Code) manejan comandos por defecto, con los que se escribe la etiqueta <head> y las etiquetas de metadatos <hmeta>, los cuales son elementos son similares para la mayoría de paginas. 

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Título</title>
            </head>
            </html>

Título <title>

la etiqueta title <title><title/> es la etiqueta que literalemete contiene el título de mi pagina web; esta etiqueta es muy importante, ya que el título es uno de los primeros elementos localizados por los buscadores, el título se muestra tal y como sale la imagen a continuación 

            <head>
                <meta charset="UTF-8">

Los metadatos <meta>

Los metadatos son datos de identificación de archivos que son interpretados tanto por el sistema operativo como por el navegador, generalmente contienen el sistema de caracteres según cada idioma 

            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
            </head>

Enlaces a archivos de estilo en cascada <link>

Los enlaces a otros archivos se traen por la etiqueta <link>, esta etiqueta trae los enlaces externos de css, los cuales tienen como función dar un aspecto estetico a la pagina web. Los enlaces se presentan como se muestra a continuación 

            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <link rel="stylesheet" type="text/css" href="cap02-ejercicio-1.css">
                <script src="cap02-ejercicio-1.js"></script>
                <script src="jquery.js"></script>
                <title>Ejercicios capitulo 2</title>
            </head>

Los enlaces de codigo interactivo javascript <script>

Los enlaces de script, traen el código que se encarga de la gestión de la interacción del usuario con la página. Javascript es un lenguaje de programación que se encarga de la interactividad; es decir, este lenguaje reacciona a las solicitudes del usuario a partir de una serie de eventos. Si pruebas escribir el siguiente código 

        <script> 
            var click = 0;    
            function incrementar()
            {
                click = click + 1;
                document.getElementById("contador").innerHTML = click;
            }
        </script> 
        <button type="button" onclick=incrementar()>aumenta</button>
        <p id="contador">0</p>

El valor resultante en el documento html será

0

En el ejemplo anterior, la función incrementar(), hace que se escriba en el documento html las veces que el usuario da click al botón llamado aumentar, específicamente en el parrafo identificado como contador <p id="contador">, haciendo que este incremento aumente de 1 en 1 click = click + 1 .En concreto, el usuario activa el evento de incrementar el numero situado en la página, y javascript responde a esa solicitud activando la función

Hay desarrolladores web que incluyen las etiquetas script en la cabecera del documento html, por lo cual esta clase de etiquetas se consideran parte de la cabecera. Sin embargo, hay otros desarrolladores que la incluyen al final de la etiqueta body, ya que esto hace más eficiente la página, cuando el navegador carga primero los elementos visuales. Sin embargo, con la rapidez de los dispositivos de hoy esta diferencia no es percibida por el usuario. En realidad, esta elección es potestativa del desarrollador web
