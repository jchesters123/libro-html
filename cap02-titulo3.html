El cuerpo del documento html

El llamado cuerpo del documento html se delimita por la etiqueta<body><body>, esta etiqueta contiene el contenido del documento; ahora bien, este contenido se estrcutura de una forma especial.

Etiquetas de estructuración

Generalmente, lo primero que hace un desarrollador web es decidir la presentación general de la página, para eso realiza un boceto de como van a ser los bloques de la pagina html, para hacer eso, se debe literalmente dividir el espacio del navegador para eso es que se usan las etiquetas de estructuración de contenidos. Por terminos generales, las etiquetas mas usadas para hacer esto son las etiquetas <nav>, <div>, <article>, entre otras

De tal forma que las etiquetas mas usadas son

Cabecera <header>

Representa la cabecera, se usa para presentar la información introductoria

Menú de navegación <nav>

Contiene el menu de navegación

Contenido <section>

Define una sección del documento

Artículo <article>

Representa contenido de texto independiente

Barra Lateral <aside>

Contiene un archivo de texto lateral a determinado contenido

Footer <footer>

Representa el cierre del documento, suele usarse para poner información de copyrgith, los créditos e información de contacto
