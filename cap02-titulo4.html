Imagenes <img>

HTML permite incorporar archivos de imagen con la etiqueta <img>, las imagenes son fundamentalmente mapas de bits que contienen pixeles, las cuales son unidades de color hay varios formatos de imagenes, los formatos de imagen que mas se usan en el desarrollo web son:

  • .jpg: Es un formato de imagen que no soporta compresiones, se usa para imagenes con muchos colores como fotos

  • .png: Es un formato de imagen que permite transparencias, cuando se comprime no tiene tanto grado de compresiones

  • .gif: Es un formato de animación que tiene una gama de 256 colores

Hay algunos desarrolladores web que realizan cursos de edición de imagen, tales como photoshop, y aplicaciones y programas gratis como GIMP. Para adjuntar ina imagen se usa la etiqueta <img>

        <img src="html.png">

Enlaces <a>

Html es un lenguaje que funciona a partir de nodos de información que son medios por los cuales el usuario puede navegar a voluntad, los enlaces permiten navegar, entre enlaces de la misma página, o a enlaces de otras páginas. Para añadir un enlace usamos la etiqueta <a></a>. Los tres posbibles enlaces que se pueden realizar en html son:

  • De dirección absoluta: cuando el enlace remite a una página web. Por ejemlplo "https://www.google.com"

  • De dirección relativa: cuando el enlace remite a un archivo localizado en el servidor. Por ejemplo "archivo/fot.png"

  • De correo electronico: cuando el enlace remite a un correo electronico. Por ejemplo "jfrancoh@ucentral.edu.co"

En los enlaces de dirección, las etiquetas generaban un hipervínculo a un enlace externo de la página web, pero también es posible generar enlaces adentro de la misma página, o elfragmento de la página web, esto es posible realizarlo de dos maneras. Utilizando el atributo name="" o el atributo id=""

Archivos multimedia de <audio> y <video>

HTML tiene etiquetas para traer archivos de audio y video

Las innovaciones en html5 permiten cargar archivos de audio y video sin tener que usar plugins, y gracias a conocimientos elementales en javascript, los desarrolladores pueden hacer muchas aplicaciones para esta etiqueta existe una etiqueta complementaria <source>. La etiqueta source da soporte a las etiquetas de audio y video para adaptar diferentes formatos. Ya que los navegador manejan diferentes
licencias de uso de archivos de metadatos de audio y video. Sin embargo, los formatos de mas licencias son mp3, mp4, wav, y h.264. La forma de usar la etiqueta source 

        <audio controls>  
            <source src="horse.ogg" type="audio/ogg">
            <source  src="horse.mp3" type="audio/mp3">
        </audio>

Además hay unos atributos que suelen ser muy usados

  • autoplay: comienza a reporducirse inmediatamente cuando carga la pagina

  • controls: da controles de reproducción a la pista de audio

  • loop: reproduce el archivo repetitivamente

a continuación, veamos la forma de enlazar un archivo de audio. Si queremos importar un archivo cualesquiera de mp3, entonces el codigo es:

        <audio controls>
            <source src="The Cranberries - Ode To My Family.mp3" type="audio/mpeg">
        </audio>

Tablas <table>

Una tabla es escencialmente una intersección de filas y columnas denominadas celdas las cuales contienen datos. Las etiquetas utilizadas para tales efectos son las etiquetas <tr> y <td>, las cuales son las etiquetas de table row o fila de la tabla tr, y table data o dato de la tabla. De tal forma que con la etiqueta <tr> se asigna una fila, y a esta fila a su ves, se le asingan las celdass que va a tener. Para poderlo tener más claro veamos el siguiente ejemplo 

                <tr>
                    <td><strong>CURSO</strong></td>
                    <td><strong>HORAS</strong></td>
                    <td><strong>HORARIO</strong></td>
                </tr>

                    <td<CSS</td>
                    <td<20</td>

                <tr/tr<
                <tr>
                    <td<HTML</td>
                    <td<20</td>
                    <td<16: 00 - 20: 00</td>
                </tr>
