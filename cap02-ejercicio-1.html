HTML:5 El Lenguaje

Breve introducción a a HTML

HTML es el lenguaje en el que se escriben las paginas web, el termino fue acuñado por el científico britanico Tim Berners Lee, que concebía al hipertexto como: "un documento el cual contiene una serie de nodos de información por los cuales el usuario puede navegar a voluntad", su ultima versión es html5, cuenta con etiquetas que permiten contener archivos de video, audio, texto, enlaces, etc

Html es un texto basado en marcas, las marcas (o tags) <marca>. Las marcas siguen la notación que se muestra a continuación

            <oscuro> este texto va en negritas </oscuro>
            y esta <importante> palabra </importante> es importante

En este ejemplo, las marcas <oscuro> e <importante> le indican al nevegador que formatos debe tener las palabras, la idea es que esas etiquetas contienen unas funciones por defecto, que le permiten identificar a los navegadores que los textos encerrados en ellas van en negrita y en énfasis correspondientmente, así es como funciona el documento html

Etiquetas y atributos

Por terminos generales, las etiquetas tienen la siguiente presentación

            <marca>contenido</marca>

Las etiquetas tienen un cierre con el /, pero tambien hay etiquetas de presentación simple, tal y como se ve en el siguiente ejemplo

            <img>                                          <img/>

Todas las etiquetas tienen atributos, los atributos contienen un conjunto de funciones que son interpretadas por un navegador

                        atributo
        
             <img src="">

            etiqueta

hay atributos de hiperenlace y atributos informativos, los atributos de hiperenlace son aquellos que permiten importar una serie de contenidos

            <img src="images/imagen.jpg">

los atributos informativos son los que permiten a los desarrolladores obtener cierta información con respecto a que hace con ciertas etiquetas

            <img alt="imagen de cabacera"/>

Etiquetas de la cabacera del documento <head>

ETIQUETA DESCRIPCIÓN
<head> Contiene las etiquetas que afectan al documento en terminos generales <meta>, <title>, <script>, <link>
<title> Da el título a la página web
<meta> contiene los elementos que son identificados tanto por el sistema operativo como por el navegador
<link> incorpora archivos externos de diseño al documento web
<script> incorpora los codigos de javscript, que se encargan de la gestión de la interacción del usuario con la página

Etiquetas ubicadas en la etiqueta <body>

Etiquetas de estructuración de contenido

ETIQUETA DESCRIPCIÓN
<header> Representa la cabecera, se usa para presentar la información introductoria
<footer> Representa el cierre del documento, suele usarse para poner información de copyrgith, los créditos e información de contacto
<nav> Contiene el menu de navegación
<section> Define una sección del documento
<article> Representa contenido de texto independiente
<figure> Contiene un archivo de imagen independiente
<aside> Contiene un archivo de texto lateral a determinado contenido
<details> Representa una serie de detalles que el usuario puede ocultar

Etiquetas de nivel de énfasis de texto

ETIQUETA DESCRIPCIÓN
<p> Se usa para caracterizar los parrafos
<h1> .... <h6> Se usa para poner títulos, donde h1 es el título con mayor enfasis y h6 el título con menor enfasis

Etiquetas de caracterización textual

ETIQUETA DESCRIPCIÓN
<br> añade un salto de renglón
<em> da un menor enfasis, tiene un atributo por defecto en el texto de cursiva
<strong> da un mayor enfasis, sirve para insertar texto de codigo en un parrafo

Listas

ETIQUETA DESCRIPCIÓN
<ol> Sirve para poner listas numeradas
<ul> Sirve para poner listas no numeradas
<dd> <dt> Se utilizan para listas de definiciones donde la etiqueta <dt> se refiere a los terminos, y la etiqueta <dd> a las definiciones de esos terminos

Los hipervínculos

ETIQUETA DESCRIPCIÓN
<a> Establece un enlace, los enlaces llevan a otra parte de la pagina o a paginas externas

Etiquetas de importación de archivos multimedia

ETIQUETA DESCRIPCIÓN
<img> Sirve para contener imagenes
<audio> Sirve para importar archivos de audio
<video> Sirve para importar archivos de video
<source> Se utiliza para soportar los formatos de las etiquetas de audio y de video

Etiquetas asociadas a tablas

ETIQUETA DESCRIPCIÓN
<table> Se usa para poner tablas
<tr> Inserta una fila adentro de la tabla, la cantidad de casillas adentro de esta etiqueta depende de la etiqueta <td>
<td> Inserta una casilla de datos adentro de la etiqueta lttr>, lo cual determina el ancho de la tabla
<thead> Es una etiqueta especial usada para poner la fila especial de una tabla
<tfoot> Es una etiqueta usada para poner la ultima fila de una tabla

Etiquetas de envío de datos del cliente al servidor

ETIQUETA DESCRIPCIÓN
<form> Los formularios se usan para partes de la pagina que solicita los datos de un usuario
<label> Es el texto que explica al cliente, la forma en la que tiene que suministrar sus datos
<input> Es el boton usado para extraer datos del cliente al servidor
